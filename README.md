abacussfx
=========

AbacusS(cala)FX - ScalaFX application to create a standalone software Abacus :)
This is a translated version of @mittie Abacus.

Inspiration
===========
This idea is inspired from Dierk König ( [@mittie](https://twitter.com/mittie) ) JavaFX example application [Learning JavaFX by example](http://www.canoo.com/blog/2013/01/08/learning-javafx-by-example/)

Prerequisite
============

* Any Scala IDE (Eclipse, NetBeans, IntelliJ etc)
* Scala-SBT
